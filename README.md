## How It Works

The website fetches random quotes from the [Quotes API](https://quotes-api-ruddy.vercel.app) and displays them with a beautiful, minimalist interface.

## Usage

1. Clone this repository:
   ```bash
   git clone https://github.com/Chronos778/quote.web.git
   ```

2. Open `index.html` in your web browser

3. Click "New Quote" to get a new random quote, or "Copy" to copy the current quote to your clipboard
